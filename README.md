## Scoreboard App

App to be used during **Startup Ping Pong** events

## Setting up

1. In project root

```
yarn install
```

2. Set a port for the backend

```
set PORT=3001
```

3. Run the server

```
nodemon bin/www
```

4. In the client folder

```
yarn install
```

5. Run the client

```
yarn start
```

## Instructions

The `Scoreboard` App display and manages information of tournaments and currently signed up teams.
